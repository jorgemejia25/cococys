Registrar un ticket

Se leen los datos del usuario con Console.ReadLine() y los guardamos en una List.

aSe solicitan al usuario el título y prioridad
bSe convierte el valor de texto a número con int.Parse
Program.cs
var tickets = new List<Ticket>(); Console.Write("Título: "); string titulo = Console.ReadLine(); Console.Write("Prioridad (1-3): "); int p = int.Parse(Console.ReadLine()); tickets.Add(new Ticket { Id = tickets.Count + 1, Titulo = titulo, Prioridad = p, Estado = "Abierto" });

El menú: unir todo

Un bucle while mantiene el programa en ejecución hasta que el usuario sale. Aquí se juntan todos los fundamentos.
